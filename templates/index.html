Сезоны

Присоединиться можно со старта любого сезона.

  • Сезон - ограниченный по времени этап лиги
  • За сезон можно сыграть 1 раз c каждым соперником
  • Могут остаться несыгранные матчи
  • Побеждает игрок с наибольшим количеством побед

Дивизионы

В течение сезона матчи только против оппонентов из своего дивизиона

  • Игроки разделены на дивизионы по 8-10 участников
  • По итогам сезона top-3 выходят в дивизион выше
  • Худшие 3 понижаются в дивизионе
  • 1, 2 места получают призы
